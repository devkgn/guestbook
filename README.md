# 프로젝트명

## 소개

### 목표
방명록 등록과 조회 기능만 있는 간단한 프로젝트를 통해 다양한 아키텍쳐의 CRUD 게시판을 구축할 수 있습니다.

### 기능
- 방명록 글 추가
- 방명록 전체 글 조회
