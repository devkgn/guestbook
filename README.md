# 프로젝트명

### 목적
방명록 등록과 조회 기능만 있는 간단한 프로젝트를 통해<br>
다양한 방식(model2, MVC, Spring Legacy, Spring Boot)로<br>
CRUD 게시판을 구축할 수 있는 연습할 할 수 있습니다. 

### 주요기능
1. 방명록 글 추가
2. 방명록 전체 글 조회
